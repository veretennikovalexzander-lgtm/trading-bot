# Trading Bot — Требования

## 1. Бизнес-требования (BR)

### BR-1. Цель проекта
Проверить жизнеспособность автоматической торговли на Binance Spot
с минимальным риском. Оценить прибыльность стратегии Mean Reversion.

### BR-2. Торговые условия

| Параметр | Значение |
|---|---|
| Биржа | Binance (тестнет → реальный спот) |
| Пары | BTC/USDT, ETH/USDT |
| Стартовый депозит | 100 USDT |
| Объём на сделку | 10% от баланса (~10 USDT) |
| Риск на сделку | 1–5% (настраивается в БД) |
| Тип торговли | Спот (без плеча) |
| Режим | Скальпинг (5-минутные свечи) |
| Максимум сделок в сутки | 30 |

### BR-3. Фиксация прибыли
После каждой прибыльной сделки излишек > 100 USDT переводится
на отдельный кошелёк / суб-аккаунт. Торговля всегда ведётся со 100 USDT.

### BR-4. Защита капитала (Circuit Breaker)

| Уровень | Условие | Действие |
|---|---|---|
| Жёлтый | ATR > 3% за час | Пауза 30 мин + уведомление |
| Оранжевый | Просадка -10% за 24ч | Стоп до ручного анализа |
| Красный | Падение цены -5% за час | Полная остановка + экстренное уведомление |
| Доп. | 3 убытка подряд | Пауза 15 мин |

### BR-5. Telegram-управление
Бот доступен только владельцу (белый список chat_id).
Команды: /start, /stop, /status, /close SYMBOL, /logs N, /config.

### BR-6. Минимизация комиссий
Использовать BNB для скидки 25%. Приоритет лимитных ордеров (мейкер).
